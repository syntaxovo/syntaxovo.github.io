☕ Java 简介

  1. Java 是一门面向对象、跨平台的编程语言。

  2. Java 语言是解释型的
    解释性语言：javascript,PHP, java
    编译性语言: c / c++
    区别是：解释性语言，编译后的代码，不能直接被机器执行,需要解释器来执行, 编译性语言, 编译后的代码, 可以直 接被机器执行。

  3. JDK，JRE

    JDK 基本介绍

    1. JDK 的全称(Java Development Kit Java 开发工具包) JDK = JRE + java 的开发工具[java, javac,javadoc,javap 等]
    2. JDK 是提供给Java 开发人员使用的，其中包含了java 的开发工具，也包括了JRE。

    JRE 基本介绍

    1. JRE(Java Runtime Environment Java 运行环境) JRE = JVM + Java 的核心类库[类]
    2. 包括Java 虚拟机(JVM Java Virtual Machine)和Java 程序所需的核心类库等，如果想要运行一个开发好的Java 程序，计算机中只需要安装JRE 即可。

    JDK、JRE 和JVM 的包含关系

    1. JDK = JRE + 开发工具集（例如Javac,java 编译工具等)
    2. JRE = JVM + Java SE 标准类库（java 核心类库）
    3. 如果只想运行开发好的.class 文件只需要JRE

🧪 Java 类型声明

每种数都定义了明确的数据类型，在内存中分配了不同大小的内存空间

  • 基本类型：
    数值型： 整型 ： byte[1] short[2] int[4] long [8]
    浮点数： float[4] double[8]
    字符型： chart[2] 存放单个字符 ‘a’
    布尔型： boolean 存放 true,false

  • 引用数据类型：
    类 class
    接口 interface
    数组 []

  • 使用事项:

  1. 变量表示内存中的一个存储区域[不同变量，类型不同，占用的空间大小不同]
  2. 该区域有自己的名称[变量名] 和类型 [数据类型]
  3. 变量必须先生命后使用
  4. 该区域的数据可以在统一类型范围内不断变化
  5. 变量在同一个作用域内不能重名
  6. 当左右两边都是数值型则做加法运算
  7. 左右两边有一方为字符型，则做拼接运算

🧪 浮点数

 * 浮点数
 * 单精度 float  占用4字节 
 * 双精度 double 占用8字节 
 * 浮点数的表示形式有两种
 * 	  Java 的浮点型默认为doube 类型，声明是float 型常量是需要加 f或者F
 * 	 浮点数在机器中存放形式： 符号位 + 指数位 + 尾数位
 * 		十进制形式 如 5.12   .512 512.0f
 * 		科学计数法 5.12e2  5.12E-2
 * 	 通常情况下应该使用double形式，他比float 型更加准确
 *  double num9 = 2.122312312312
 *  float  num10 = 2.122312312312F
 *  尾数部分可能丢失，造成精度丢失

🧪 自动转换

  • java 程序在进行赋值或者运算是，精度小的类型会自动转换为精度大的数据类型称为自动类型转换

    数据类型按精度从小到大排序

    bite[1] -> short[2] -> int[4] -> long[8] -> float[4] -> double [8]

    char[2] -> int[4] -> long[8] -> float[4] -> double[8]

    规则：

    1、有多种类型的数据混合运算时，系统首先自动将所有数据转为容量最大的那种数据类型，然后进行计算

    2、把精度大的数据类型赋值给精度小的会报错，反之会自动进行类型转换

    3、(byte,short) 和char 之前不会进行相互自动转换，把具体数值赋值给byte时，先判断该数是否在byte范围内，是就可以

    4、byte、short、char 三者可以进行计算，只要存在一种类型都会转为int 类型参与运算

    5、boolean 不参与转换

    6、自动提升，表达式结果的类型自动提升为操作数中最大的类型

强制转换

  • 自动转换的逆过程中，将容量大的数据类型转为容量小的数据类型，使用时要加上强制转换符（）。
  • 当进行数据转换时，从大到小就需要用到强制转换
  • 强转符合只针对最近的操作数有效，往往会使用（）提升优先级
  • char 类型可以保存int 的常量值,但不能保存变量值
  • 5.byte 和 short 类型在进行运算时当作int 类型处理

String

  • 在程序开发过程中，我们经常需要将基本数据类型转为String 类型或者将String 转为 基本数据类型
  • 1.1 基本数据类型转为String
  • 将基本数据类型 + "" 即可
  • 1.2 String 转为基本数据类型
  • 通过基本类型的包装类调用parseXX方法即可
  • 1.3 在将String 类型转为基本数据类型时，要确保String 类型能够转为有效的数据；
  • 如果格式不正确，就会抛出异常，程序终止
